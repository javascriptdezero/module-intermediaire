En cas de problèmes techniques

Il se peut que vous rencontriez certains problèmes techniques pour valider les exercices.
Lisez leur solution au moins une fois avant de commencer les exercices.

  • Si certains exercices précédemment validés en vert redeviennent gris et "En attente de code à tester" : c'est qu'il y a une erreur de syntaxe dans votre code. Solution : ouvrez la console développeur pour trouver l'erreur.

    Cliquez ici pour savoir comment ouvrir la console développeur de Chrome

    Sur Windows ou Linux : depuis Chrome, tapez Ctrl + Shift + J ou allez dans le menu (les 3 petits points verticaux en haut à droite de la fenêtre de Chrome) puis allez dans Plus d'outils > Outils de développement.

    Sur MacOS : depuis Chrome, tapez ⌘ + ⌥ + J (Commande + Option + J) ou bien allez dans le menu Afficher > Options pour les développeurs > Console JavaScript.

    Le message d'erreur apparaît en rouge et est en anglais, il faudra le traduire. L'erreur vient du fichier noté sur la droite au numéro de ligne indiqué. Ouvrez ce fichier dans VSCode, rendez-vous à la ligne indiquée et corrigez l'erreur.

  • Si la page web ne répond plus (les clics n'ont aucun effet) ou si l'icône dans l'onglet tourne indéfiniment : c'est qu'il y a une boucle infinie dans votre code. Solution : corrigez votre code puis actualisez le site web.

    Cliquez ici pour voir comment corriger une boucle infinie

    Si vous avez fait une boucle infinie, l'icône de votre onglet va tourner comme ça indéfiniment :

    Voici à quoi peut ressembler une boucle infinie dans du code :

    let poidsClient = prochainClient.mesurerPoids();

    while (poidsClient !== 0) {
      console.log(poidsClient);
    }

    Le problème ici, c'est que la condition de répétition de la boucle poidsClient !== 0 est toujours vraie ! En effet, on ne modifie pas la valeur de poidsClient à l'intérieur de la boucle, par conséquent celle-ci s'exécutera sans jamais s'arrêter : vous avez fait une boucle infinie !

    Voici les étapes à respecter pour se sortir de ce problème :

    1. Dans le menu de Chrome (les 3 petits points verticaux en haut à droite de la fenêtre de Chrome), allez dans Plus d'outils > Gestionnaire de tâches, sélectionnez l'onglet qui pose problème dans la liste puis cliquez sur le bouton Arrêter le processus en bas à droite.
      Vous pouvez également faire un clic droit sur la barre de titre de Chrome pour sélectionner plus rapidement le gestionnaire de tâches.

    2. Corrigez le bogue dans votre code depuis VSCode.
    3. Actualisez à nouveau la page web qui a été précédemment arrêtée en cliquant sur la petite flèche à gauche de la barre d'adresse.

    Attention à la sauvegarde automatique dans VSCode !

    Dans les paramètres, je vous conseille d'utiliser l'option onFocusChange pour la sauvegarde automatique.

    En effet, si vous avez l'option afterDelay de sélectionnée, ça veut dire que VSCode va sauvegarder votre code toutes les X secondes. Le site web va se recharger automatiquement à ce moment là et si votre code actuel génère une boucle infinie, Chrome va planter !

    Pour éviter ce problème, utilisez l'option onFocusChange qui indique à VSCode de sauvegarder votre fichier seulement si vous changez de fenêtre ou d'onglet. Vous pouvez aussi sauvegarder manuellemment avec ⌘ + S sur MacOS ou Ctrl + S sur Windows ou Linux quand vous voulez vérifier que votre code fonctionne.

  • Si vous souhaitez déboguer votre code, utilisez le débogueur de Chrome comme je l'explique dans cette vidéo.

  • Si vous êtes bloqué et qu'aucune solution mentionnée ne fonctionne, envoyez-moi un email sur jeremy@javascriptdezero.com en précisant le numéro de la leçon sur laquelle vous travaillez et le numéro de l'exercice qui pose problème.

Structure des exercices

Exercices d'échauffement

Dans un premier temps, vous allez devoir utiliser les méthodes listées ci-dessus une par une, dans l'ordre, pour réaliser les exercices d'échauffement. Vous ferez quelques exercices sur concat(), puis quelques exercices sur fill(), puis includes() etc.

Même si ces exercices peuvent parfois être un peu rébarbatifs, faites-les au moins une fois. L'objectif est de vous familiariser avec chacune de ces méthodes pour retenir l'utilité principale de celles-ci.

Exercices finaux

Une fois l'échauffement terminé, les exercices suivants vont devenir un peu plus difficiles : vous devrez utiliser plusieurs des méthodes vues précédemment pour répondre à l'énoncé.

Indices

Pour les exercices finaux vous pourrez afficher un indice qui vous indiquera quelles méthodes utiliser pour répondre à l'énoncé. N'affichez l'indice que si vous avez passé au moins 30 min sur un exercice. Le but n'est pas d'aller le plus vite possible mais de faire un maximum d'exercices seul.

L'objectif est de vous apprendre à utiliser les nouveaux outils à votre disposition (les méthodes des tableaux) pour répondre à des exercices de plus en plus complexes.

Vous développerez votre sens logique et votre capacité à assembler des briques de base (les méthodes) entre elles pour répondre à des énoncés complexes.

Fonctionnement des exercices

Tous les exercices fonctionnent sur le même principe : vous devez rédiger une fonction à partir de zéro et celle-ci devra renvoyer (avec le mot-clé return) la solution de l'énoncé.

Exercice 0

Créez une fonction concat0(tableau1, tableau2) qui renvoie la concaténation des tableaux tableau1 et tableau2 dans cet ordre.

Exemple : concat0([1, 2, 3], [4, 5, 6]) renverra un tableau dont la valeur sera [1, 2, 3, 4, 5, 6].

Exercice 1

Créez une fonction concat1(tableau, valeur1, valeur2, valeur3) qui renvoie la concaténation de tableau et valeur1, valeur2 et valeur3 dans cet ordre.

Exemple : concat1([1, 2, 3], 4, 5, 6) renverra un tableau dont la valeur sera [1, 2, 3, 4, 5, 6].

Exercice 2

Créez une fonction fill0(tableau, valeur) qui renvoie le tableau dont on aura remplacé tous les éléments par la valeur valeur.

Exemple : fill0([1, 2, 3], 0) renverra un tableau dont la valeur sera [0, 0, 0].

Exercice 3

Créez une fonction fill1(tableau, valeur, index) qui renvoie le tableau dont on aura remplacé tous les éléments par la valeur valeur à partir de l'index index et jusqu'à la fin du tableau.

Exemple : fill1([1, 2, 3], 0, 1) renverra un tableau dont la valeur sera [1, 0, 0].

Exemple : fill1(['a','b','c','d','e'], 'z', 3) renverra un tableau dont la valeur sera ['a','b','c','z','z'].

Exercice 4

Créez une fonction fill2(tableau, valeur, debut, fin) qui renvoie le tableau dont on aura remplacé tous les éléments par la valeur valeur à partir de l'index debut et jusqu'à l'index fin (inclus, attention, j'ai bien dit inclus !).

Exemple : fill2([1, 2, 3, 4, 5], 0, 1, 3) renverra un tableau dont la valeur sera [1, 0, 0, 0, 5].

Exercice 5

Créez une fonction fill3(tableau, valeur) qui renvoie le tableau dont on aura remplacé les 3 derniers éléments par la valeur valeur.

Exemple : fill3([1, 2, 3, 4, 5], 42) renverra un tableau dont la valeur sera [1, 2, 42, 42, 42].

Exemple : fill3(['a', 'b', 'c', 'd', 'e', 'f', 'g'], 'z') renverra un tableau dont la valeur sera ['a', 'b', 'c', 'd', 'z', 'z', 'z'].

Indice

Que se passe-t-il si vous utilisez une valeur négative pour l'index de début de fill() ?

Exercice 6

Créez une fonction includes0(tableau, valeur) qui renvoie true si tableau contient la valeur valeur et false sinon.

Exemple : includes0([1, 2, 3, 4], 2) renverra true.

Exemple : includes0(['a', 'b', 'c', 'd', 'e'], 'z') renverra false.

Exercice 7

Créez une fonction includes1(tableau, valeur, debut) qui renvoie true si tableau contient la valeur valeur à partir de l'index debut (jusqu'à la fin du tableau) et false sinon.

Exemple : includes0([1, 2, 3, 4], 3, 1) renverra true.

Exemple : includes0(['a', 'b', 'c', 'd', 'e'], 'b', 2) renverra false.

Exercice 8

Créez une fonction includes2(tableau, valeur) qui renvoie true si tableau contient la valeur valeur dans les 4 derniers éléments du tableau et false sinon.

Exemple : includes0([1, 2, 3, 4, 5, 6], 5) renverra true.

Exemple : includes0(['a', 'b', 'c', 'd', 'e', 'f'], 'a') renverra false.

Exercice 9

Créez une fonction indexOf0(tableau, valeur) qui renvoie l'index de la valeur valeur dans le tableau tableau si elle est présente dans celui-ci et -1 sinon.

Exemple : indexOf0([1, 2, 3, 4, 5, 6], 5) renverra 4.

Exemple : indexOf0(['a', 'b', 'c', 'd', 'e', 'f'], 'z') renverra -1.

Exercice 10

Créez une fonction indexOf1(tableau, valeur, debut) qui renvoie l'index de la valeur valeur dans le tableau tableau si elle est présente dans celui-ci à partir de l'index debut du tableau (jusqu'à la fin du tableau) et -1 sinon.

Exemple : indexOf1([1, 2, 3, 4, 5, 6], 3, 2) renverra 2.

Exemple : indexOf1(['a', 'b', 'c', 'd', 'e', 'f'], 'd', 4) renverra -1.

Exercice 11

Créez une fonction indexOf2(tableau, valeur) qui renvoie l'index de la valeur valeur dans le tableau tableau si elle est présente dans les 2 derniers éléments de celui-ci et -1 sinon.

Exemple : indexOf2([1, 2, 3, 4, 5, 6], 5) renverra 4.

Exemple : indexOf2(['a', 'b', 'c', 'd', 'e', 'f'], 'd') renverra -1.

Exercice 12

Créez une fonction join0(tableau) qui renvoie une chaîne de caractères représentant la concaténation des éléments du tableau séparés par une virgule.

Exemple : join0(["pierre", "papier", "ciseaux"]) renverra "pierre,papier,ciseaux".

Exemple : join0([1,2,3,4]) renverra "1,2,3,4".

Exercice 13

Créez une fonction join1(tableau, separateur) qui renvoie une chaîne de caractères représentant la concaténation des éléments du tableau séparés par la valeur de separateur.

Exemple : join1(["pierre", "papier", "ciseaux"], " ou ") renverra "pierre ou papier ou ciseaux".

Exemple : join1([1,2,3,4], " ") renverra "1 2 3 4".

Exercice 14

Créez une fonction lastIndexOf0(tableau, valeur) qui renvoie l'index du dernier élément valeur du tableau ou -1 si l'élément n'est pas présent.

Exemple : lastIndexOf0(['a','b','c','a','d'], 'a') renverra 3.

Exemple : lastIndexOf0([1,2,3,4,1,2,3,4], 2) renverra 5.

Exemple : lastIndexOf0([1,2,3,4,1,2,3,4], 12) renverra -1.

Exercice 15

Créez une fonction lastIndexOf1(tableau, valeur) qui renvoie l'index du dernier élément valeur contenu dans les 4 premiers éléments du tableau ou -1 si l'élément n'est pas présent.

Exemple : lastIndexOf1(['a','b','c','a','a'], 'a') renverra 3.

Exemple : lastIndexOf1([1,2,2,4,2,2,2], 2) renverra 2.

Exemple : lastIndexOf1([1,2,3,4,1,2,3,4], 12) renverra -1.

Exercice 16

Créez une fonction lastIndexOf2(tableau, valeur) qui renvoie l'index du dernier élément valeur contenu dans le tableau sans prendre en compte les 2 derniers éléments du tableau ou -1 si l'élément n'est pas présent.

Exemple : lastIndexOf1(['a','b','c','a','b'], 'a') renverra 0.

Exemple : lastIndexOf1([1,2,2,4,2,2,2], 2) renverra 4.

Exemple : lastIndexOf1([1,2,3,4,1,2,3,4], 12) renverra -1.

Exercice 17

Créez une fonction pop0(tableau) qui renvoie le dernier élément de tableau.

Exemple : pop0(['a','b','c','d','e']) renverra "e".

Exemple : pop0([1,2,3,4]) renverra 4.

Exercice 18

Créez une fonction pop1(tableau) qui renvoie le tableau sans son dernier élément.

Exemple : pop1(['a','b','c','d','e']) renverra ['a','b','c','d'].

Exemple : pop1([1,2,3,4]) renverra [1,2,3].

Exercice 19

Créez une fonction push0(tableau, valeur) qui ajoute valeur à la fin de tableau et renvoie celui-ci.

Exemple : push0(['a','b','c','d','e'], 'f') renverra ['a','b','c','d','e','f'].

Exemple : push0([1,2,3,4], 5) renverra [1,2,3,4,5].

Exercice 20

Créez une fonction push1(tableau, valeur1, valeur2) qui ajoute valeur1 et valeur2 à la fin de tableau et renvoie celui-ci.

Exemple : push1(['a','b','c','d','e'], 'f', 'g') renverra ['a','b','c','d','e','f','g'].

Exemple : push1([1,2,3,4], 5, 6) renverra [1,2,3,4,5,6].

Exercice 21

Créez une fonction reverse0(tableau) qui renvoie tableau avec tous ses éléments dans l'ordre inverse.

Exemple : reverse0(['a','b','c','d','e']) renverra ['e','d','c','b','a'].

Exemple : reverse0([1,2,3,4]) renverra [4,3,2,1].

Exercice 22

Créez une fonction shift0(tableau) qui renvoie le premier élément de tableau.

Exemple : shift0(['a','b','c','d','e']) renverra "a".

Exemple : shift0([1,2,3,4]) renverra 1.

Exercice 23

Créez une fonction shift1(tableau) qui renvoie le tableau sans son premier élément.

Exemple : shift1(['a','b','c','d','e']) renverra ['b','c','d','e'].

Exemple : shift1([1,2,3,4]) renverra [2,3,4].

Exercice 24

Créez une fonction slice0(tableau, debut, fin) qui renvoie un tableau contenant tous les éléments du tableau à partir de l'index debut jusqu'à la fin du tableau.

Exemple : slice0(['a','b','c','d','e'], 2) renverra ['c','d','e'].

Exemple : slice0([1,2,3,4], 2) renverra [3,4].

Exercice 25

Créez une fonction slice1(tableau, debut, fin) qui renvoie un tableau contenant tous les éléments du tableau à partir de l'index debut jusqu'à l'index fin (exclus).

Exemple : slice1(['a','b','c','d','e'], 1, 4) renverra ['b','c','d'].

Exercices finaux

Exercice 0 — Demi-tour

Créez une fonction demitour(tableau1, tableau2) qui renvoie un tableau contenant tous les éléments de tableau1 et tableau2 à l'envers.

Exemple : demitour([1,2,3], ['a','b','c']) renverra ['c','b','a',3,2,1].

Indice

Utilisez les méthodes reverse() et concat().

Exercice 1 — Remplissage

Créez une fonction remplissage(mot, taille) qui renvoie une chaîne de caractères de longueur taille contenant les caractères de mot précédés du caractère #.

Exemple : remplissage(['c','o','d','e'], 8) renverra ####code.

Indice

Utilisez les méthodes unshift() et join().

Exercice 2 — Extraction

Créez une fonction extraction(transmission, sequence) qui renvoie un tableau contenant le code secret du tableau transmission. Ce code secret est compris entre les 2 éléments de valeur sequence.

Il est garanti qu'il n'existe qu'un seul code secret dans la transmission.

Il est garanti qu'il existe exactement 2 éléments ayant la valeur sequence dans le tableau transmission.

Exemple : extraction(['a','r','#','c','o','d','e','#','g','h','t'], '#') renverra ['c','o','d','e'].

Indice

Utilisez les méthodes indexOf() et lastIndexOf() avec slice().

Exercice 3 — Camouflage

Créez une fonction camouflage(transmission, sequence) qui renvoie un tableau contenant tous les éléments du tableau transmission mais dont le code secret aura été camouflé par le caractère #.

Il est garanti qu'il n'existe qu'un seul code secret dans la transmission.

Il est garanti qu'il existe exactement 2 éléments ayant la valeur sequence dans le tableau transmission.

Exemple : camouflage(['a','r','#','c','o','d','e','#','g','h','t'], '#') renverra ['a','r','#','#','#','#','#','#','g','h','t'].

Indice

Utilisez les méthodes indexOf() et lastIndexOf() avec fill().

Exercice 4 — Suppression

Créez une fonction suppression(transmission, sequence) qui renvoie un tableau contenant tous les éléments du tableau transmission mais dont le code secret aura été supprimé.

Il est garanti qu'il n'existe qu'un seul code secret dans la transmission.

Il est garanti qu'il existe exactement 2 éléments ayant la valeur sequence dans le tableau transmission.

Exemple : suppression(['a','r','#','c','o','d','e','#','g','h','t'], '#') renverra ['a','r','#','#','g','h','t'].

Indice

Utilisez les méthodes indexOf() et lastIndexOf() avec splice().

Exercice 5 — Entrelacement

Créez une fonction entrelacement(tableau1, tableau2) qui renvoie un tableau contenant tous les éléments du tableau1 et du tableau2 entrelacés un sur deux.

Il est garanti que tableau1 et tableau2 auront toujours la même longueur.

Exemple : entrelacement(['a','b','c'], [1,2,3]) renverra ['a',1,'b',2,'c',3].

Indice

Utilisez une boucle à l'aide de la méthode push() pour construire le tableau final.

Exercice 6 — Duplication

Créez une fonction duplication(quantites, valeurs) qui renvoie un tableau contenant chaque valeur du tableau valeurs dupliquées N fois. Le nombre de copies N à faire est indiqué dans le tableau quantites.

Il est garanti que quantites et valeurs auront toujours la même longueur.

Exemple : duplication([1,2,3], ['a','b','c']) renverra ['a','b','b','c','c','c'].

Indice

Utilisez une boucle imbriquée (une boucle dans une boucle). Boucle 1 : lisez les valeurs quantites une par une, pour chacune d'entre elles, ajoutez plusieurs fois (boucle 2) à l'aide de la méthode push() la valeur correspondante pour construire le tableau final.

Exercice 7 — Sélection

Créez une fonction selection(selecteur, liste) qui renvoie un tableau de résultats contenant certains éléments de liste dans un certain ordre. Voici comment procéder pour créer ce tableau de résultats.

Pour chaque élément dans selecteur :

  • Si celui-ci se trouve dans le tableau liste, on ajoutera au tableau de résultats le premier élément de liste. Ce premier élément sera supprimé de liste.
  • Si celui-ci ne se trouve pas dans le tableau liste on ajoutera au tableau de résultats le dernier élément de liste. Ce dernier élément sera supprimé de liste.
  • À chaque fois qu'on passe au sélecteur suivant la liste aura rétrécit car on lui retire soit le premier élément, soit le dernier !

Exemple : selection([1,2,3,4,5], [10,20,3,6,70,50,4]) renverra [4,50,10,70,6].

Détails des étapes :

  1. Est-ce que 1 est dans [10,20,3,6,70,50,4] ?
  2. Non, alors on ajoute le dernier élément 4 au tableau de résultats et on retire cet élément de la liste !
  3. Est-ce que 2 est dans [10,20,3,6,70,50] ?
  4. Non, alors on ajoute le dernier élément 50 au tableau de résultats et on retire cet élément de la liste !
  5. Est-ce que 3 est dans [10,20,3,6,70] ?
  6. Oui, alors on ajoute le premier élément 10 au tableau de résultats et on retire cet élément de la liste !
  7. Est-ce que 4 est dans [20,3,6,70] ?
  8. Non, alors on ajoute le dernier élément 70 au tableau de résultats et on retire cet élément de la liste !
  9. Est-ce que 5 est dans [20,3,6] ?
  10. Non, alors on ajoute le dernier élément 6 au tableau de résultats et on retire cet élément de la liste !
  11. Le résultat est donc [4,50,10,70,6].
Indice

Utilisez les méthodes includes(), shift() et pop().

Exercice 8 — Extraction multiple (difficile)

Créez une fonction extractionMultiple(transmission, sequence) qui renvoie un tableau contenant tous les codes secrets extraits du tableau transmission. Un code secret est compris entre 2 éléments de valeur sequence.

Il peut y avoir 0, 1 ou plusieurs codes secrets dans la transmission.

S'il n'y a aucun code secret, renvoyez un tableau vide. S'il y a plusieurs codes secrets, cumulez-les dans le tableau renvoyé.

Il est garanti que les codes secrets sont toujours complets, c'est-à-dire qu'il existe toujours un élément de valeur sequence avant et après le code secret qu'il faut extraire.

Exemples :

  • extractionMultiple(['a','r','c','o','d','e','t'], '#') renverra [].
  • extractionMultiple(['a','r','#','c','o','d','e','#','g','h','t'], '#') renverra ['c','o','d','e'].
  • extractionMultiple(['a','#','c','o','#','r','#','d','e','#','g'], '#') renverra ['c','o','d','e'].
Indice

Utilisez la méthode indexOf() avec un index de début qui vous permettra de chercher le prochain caractère sequence. Utilisez slice() pour extraire le code secret et concat() pour le cumuler au résultat.

Exercice 9 — Extraction multiple incomplète (difficile)

Créez une fonction extractionMultipleIncomplete(transmission, sequence) qui renvoie un tableau contenant tous les codes secrets extraits du tableau transmission. Un code secret est compris entre 2 éléments de valeur sequence.

Il peut y avoir 0, 1 ou plusieurs codes secrets dans la transmission.

S'il n'y a aucun code secret, renvoyez un tableau vide. S'il y a plusieurs codes secrets, cumulez-les dans le tableau renvoyé.

Cette fois-ci, la transmission a pu être coupée en plein milieu d'un code. Le dernier code secret n'a donc peut-être pas sa séquence de fin ! Il faudra donc extraire ce code secret jusqu'à la fin de la transmission si c'est le cas.

Exemples :

  • extractionMultipleIncomplete(['a','#','c','o','d'], '#') renverra ['c','o','d'].
  • extractionMultipleIncomplete(['a','#','c','o','#','g','#','d'], '#') renverra ['c','o','d'].
  • extractionMultipleIncomplete(['a','#','c','o','#','r','#','d','e','#','#'], '#') renverra ['c','o','d','e'].
  • extractionMultipleIncomplete(['#','c','o','#','r','#','d','e'], '#') renverra ['c','o','d','e'].

Vous pouvez repartir du code de l'exercice précédent, mais attention il va provoquer une boucle infinie pour extractionMultipleIncomplete(['a','#','c','o','d'], '#')

Indice

Même chose que l'exercice précédent mais il faut vérifier s'il existe un caractère sequence de fin, pour ça il faut vérifier si indexOf() renvoie -1 !
